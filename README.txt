○遊び方
〜アイコン〜
・アイコンは同じ種類が3つ以上繋がると消えます。
・アイコンの種類は「きらら＆ミラク」「MAX」「フォワード」「キャラット」「七賢者&クロモン」の5種類です。
・設定から出現する作品キャラを選ぶことができます。

〜消し方〜
・アイコンをタッチするとその周辺にある複数のアイコンが同時に消えます。
・ともに消えるアイコンはNEXT枠の一番手前にいるキャラクターによって変化します。
　ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー
　ポルカ：周囲8マス
　コルク：十字
　ライネ；横一列
　カンナ：縦一列＋左右
　ランプ：同種類のアイコン
　きらら：同種類のアイコン＋しばらく同種類のアイコンが出現しない
　ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー

〜ボーナス〜
・連鎖数によって獲得スコアが増加します。
・連鎖数が大きいとランプときららが出現しやすくなります。
・「七賢者&クロモン」を3つ以上繋げて消すと、VS枠にいる七賢者の体力が減少します。
・七賢者の体力が0になると撃破ボーナススコアを獲得でき、次の七賢者が出現します。
・七賢者の撃破後は特定の連鎖数に到達するまで「七賢者&クロモン」は出現しません。

〜最後に〜
・クレアちゃんは可愛い。
